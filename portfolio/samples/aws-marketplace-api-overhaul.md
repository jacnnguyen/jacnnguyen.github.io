# AWS Marketplace API Documentation Overhaul

## Overview
This was a multi-month initiative to consolidate, restructure, and standardize AWS Marketplace API documentation into a single authoritative reference.

## My Role
- Lead Technical Writer
- Owned information architecture and content strategy
- Coordinated with engineers, product managers, and an external vendor

## What I Did
- Reorganized fragmented API documentation
- Defined consistent structure and naming conventions
- Edited and published final reference content
- Ensured accuracy across numerous API endpoints

## Audience
- Developers integrating with AWS Marketplace APIs
- Enterprise partners
- Internal AWS stakeholders

## Tools & Technologies
- Markdown
- Git-based documentation workflows
- REST APIs
- Vendor collaboration tooling

## Impact
- Improved discoverability and usability of API documentation
- Created a single source of truth for developers
- Reduced confusion caused by duplicated or outdated content
